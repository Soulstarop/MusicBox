# MusicBox
Music plugin for Pocketmine Servers

# HOW TO USE
 * Install it on your server
 * Place some Noteblock
 
# COMMANDS
 * /music <start|stop|next>
 
# HOW TO MAKE SONG
 * Convert your midi file into nbs file
 * ->  http://www.stuffbydavid.com/mcnbs
